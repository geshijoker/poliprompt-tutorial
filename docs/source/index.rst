Welcome to PoliPrompt's documentation!
===================================

**PoliPrompt** (/pɒl.i'prɒmpt/) is a Python library for political analysis
with the assistance of advanced foundation models (LLM and MLLM).
The source code is at `PoliPrompt <https://github.com/geshijoker/PoliPrompt/>`_
and here we offer a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
